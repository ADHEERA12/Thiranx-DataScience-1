Project Summary – Real-World Data Analysis on Titanic Dataset

In this project, I worked on the Titanic dataset to perform real-world data analysis using Python. The project focused on understanding passenger survival patterns through data cleaning, visualization, and exploratory data analysis (EDA).

I started by importing and analyzing the dataset using Pandas. Missing values in columns such as Age and Embarked were handled using median and mode techniques. After preprocessing the data, I performed statistical analysis to understand the distribution of different variables.

Using Matplotlib and Seaborn, I created multiple visualizations including survival count plots, gender distribution charts, passenger class analysis, age distribution histograms, fare distribution graphs, and correlation heatmaps. These visualizations helped identify important trends and relationships within the dataset.

The analysis revealed several key insights:

Female passengers had a significantly higher survival rate than male passengers.
First-class passengers were more likely to survive compared to lower classes.
Passenger class and fare had a strong influence on survival chances.
Most passengers belonged to the age group between 20 and 40 years.
